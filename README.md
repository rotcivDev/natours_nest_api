# Natours NestJS API

## Description

[NodeJs Course](https://www.udemy.com/course/nodejs-express-mongodb-bootcamp/) basic project, enhanced with NestJS framework.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
